## 실습 미션: 내 링크 모음 페이지

### 왜 이 미션인가
- 2시간 안에 완성 가능
- 비전공자도 "내가 쓸 수 있는" 결과물
- Vercel 배포로 실제 URL이 생김 → 성취감 최대
- HTML/CSS 지식 0이어도 Cursor가 다 만들어줌

### 결과물 예시
```
https://내이름-links.vercel.app
- 내 이름 / 한 줄 소개
- 인스타, 유튜브, 노션, 포트폴리오 링크 모음
- 배경색, 폰트 본인이 원하는 스타일로
```

### 실습 흐름

**Step 1 — Cursor에서 프로젝트 시작 (20~35분)**
```
프롬프트 예시:
"내 링크 모음 페이지를 만들어줘.
이름은 홍길동, 직업은 디자이너.
링크: 인스타 @hong, 노션 notion.so/hong
배경은 연한 베이지, 폰트는 둥글둥글하게"
```

**Step 2 — 수정 반복 (35~70분)**  
마음에 안 드는 부분을 말로 수정 요청. 예:
- "버튼을 더 크게"
- "배경색을 민트로"
- "링크 아이콘 추가해줘"

**Step 3 — Vercel 배포 (70~95분)**
1. GitHub 계정 연결
2. 프로젝트 import
3. Deploy 클릭
4. URL 확인

## 운영자 노트

**막히는 포인트 TOP 3**
1. Cursor 로그인/API 키 — prep.md 스크린샷 보고 해결

- URL은 수업 후 카톡으로 배포 방법 영상 링크 발송

**분위기 포인트**
- 완성본 먼저 보여주는 게 첫 10분의 핵심 — "나도 저거 만들 수 있나?" 가 생겨야 2시간 집중함

- 사진은 URL 뜨는 순간 바로 찍기 — 표정이 가장 좋을 때
