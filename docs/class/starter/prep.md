# 사전 준비 가이드 — 수업 전날까지 완료

수업 당일 설치 때문에 시간 낭비 없도록 아래 3가지를 미리 해주세요.  
막히는 부분은 카톡으로 문의해도 됩니다.

---

## 1. Cursor 설치

1. [cursor.com](https://cursor.com) 접속
2. Download 클릭 → 본인 OS에 맞게 설치 (Mac / Windows)
3. 설치 후 실행 — 로그인 화면 나오면 OK

---

## 2. Cursor AI 연결 (둘 중 하나)

### 방법 A — Cursor Pro 구독 (추천, 월 $20)
1. Cursor 실행 후 우측 상단 Sign In
2. GitHub 또는 Google 계정으로 가입
3. Settings → Billing → Pro 플랜 구독
4. AI 응답이 바로 나오면 연결 완료

### 방법 B — Claude API 키 사용 (무료 크레딧 있음)
1. [console.anthropic.com](https://console.anthropic.com) 접속 → 가입
2. API Keys → Create Key → 키 복사
3. Cursor → Settings → Models → API Key에 붙여넣기
4. 모델 목록에 Claude가 보이면 연결 완료

> 어느 쪽이든 수업 중 AI 응답이 나오면 됩니다. 방법 A가 설정이 더 간단합니다.

---

## 3. 계정 2개 미리 만들기

### GitHub
1. [github.com](https://github.com) 접속 → Sign up
2. 이메일 인증까지 완료

### Vercel
1. [vercel.com](https://vercel.com) 접속 → Sign up
2. GitHub 계정으로 연결 (같은 계정 사용)
3. 대시보드 화면이 보이면 완료
