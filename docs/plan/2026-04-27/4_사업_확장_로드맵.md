# 리포트 4 — 사업 확장 로드맵

> **이 리포트의 질문**: "강의 → 스터디 → 모각작 → 커뮤니티 → 사업으로 어떻게 확장하나? 매출은 언제 어디서 나오나?"

작성일: 2026-04-27
선행 문서: 1차 기획안 §6 (수익 다각화), §7 (1차 출시 우선순위)

## 0. 한 줄 요약

> **3개 페이즈 × 4분기 = 12개월 로드맵**.
> Q1 단발 강의로 페르소나 검증 → Q2 6주 패키지 + 멤버십 베타 → Q3 영상강의·전자책으로 비강사 매출 + B2B 첫 출강 → Q4 강사 1명 추가·커뮤니티 자율운영. 12개월 끝 월 매출 500~800만, 순수익 350~500만이 목표선.
> 이 트랙을 회사 다니면서 "주말+야간"으로 굴리고, 12개월 끝 시점에 탈주 여부 결정 (리포트 5).

## 1. 매출 모델 — 6개 채널의 우선순위

| # | 채널 | 도입 시점 | 평균 회당/월 매출 | 강사 시간 부담 | 비고 |
| --- | --- | --- | --- | --- | --- |
| 1 | 단발 라이트 클래스 | Phase 0 (지금) | ₩150,000~₩200,000/회 | 회당 8~10시간 | 현금흐름의 시작 |
| 2 | **6주 패키지 (미니홈피)** | Phase 1 (5월) | ₩900,000~₩1,200,000/기수 | 기수당 30~40시간 | **메인 매출 채널** |
| 3 | 월 멤버십 구독 | Phase 2 (3개월 후) | ₩300,000~₩900,000/월 | 월 5~10시간 | LTV 핵심 |
| 4 | 영상강의 / 전자책 | Phase 3 (6개월 후) | ₩200,000~₩500,000/월 | 초기 100시간, 이후 0 | 비강사 매출 |
| 5 | B2B 출강 (회사 정기 워크숍) | Phase 3 (6~9개월 후) | ₩700,000~₩2,000,000/회 | 회당 15시간 | 천장 깨는 채널 |
| 6 | 1:1 코칭 / 멘토링 | Phase 4 (9개월 후) | ₩300,000~₩500,000/월 | 회당 1.5시간 | 멤버십 회원 한정 |

### 1-1. 채널 도입 순서의 논리
1. **단발 → 패키지**: 단발에서 페르소나·가격을 검증한 후 패키지로. 패키지를 먼저 팔면 콘텐츠 미완성 상태에서 환불 위험.
2. **패키지 → 멤버십**: 패키지 졸업생이 멤버십의 첫 고객. "졸업 후 30일 무료 멤버십" 같은 자연 전환.
3. **멤버십 → 영상**: 멤버십 라이브러리 콘텐츠가 쌓이면 → 일부를 외부 판매용 영상으로 가공.
4. **영상 → B2B**: 영상이 있으면 회사에 "이미 검증된 콘텐츠를 사내 워크숍으로" 제안 가능. 영상이 신뢰감 자산.
5. **B2B → 1:1**: B2B 클라이언트 일부가 "개인 코칭도 해줄 수 있냐" 로 자연 전환.

> **순서 어기면 망함**: 가장 흔한 실수는 멤버십을 1번부터 도입하는 것. 콘텐츠가 부족한 멤버십은 1개월 만에 회원 50% 이탈 → 신뢰 회복 6개월. 단발 → 패키지로 콘텐츠 자산을 먼저 쌓은 다음 멤버십.

## 2. 12개월 로드맵 — Q1~Q4

### Q1 (1~3개월차) — 페르소나 / 가격 / 운영의 첫 검증

**목표**: 첫 강의 굴리기. 사이트 미완성 OK.

#### 1개월차 (5월) — 첫 단발 강의
- [ ] 소모임 앱 / 당근 / 인스타에서 모임 개설
- [ ] 단발 라이트 클래스 1~2회 운영 (₩30,000 × 6명)
- [ ] 후기·결과물 캡쳐 ≥ 5건 확보
- [ ] 1차 데모 사이트 5~10명에게 보여주고 인터뷰 답변 수집

**예상 매출**: ₩300,000~₩500,000

#### 2개월차 (6월) — 6주 패키지 S1 1기수
- [ ] **S1 1기수 모집 + 운영** (5월 17일~6월 21일 토요일)
- [ ] 단발 1~2회 추가 (페르소나 다른 시간대 테스트)
- [ ] 인스타 릴스 콘텐츠 4편 (회차별 결과물 자랑)

**예상 매출**: ₩1,200,000~₩1,500,000

#### 3개월차 (7월) — S1 마무리 + S2 모집
- [ ] **S1 1기수 졸업 + 데모데이**
- [ ] **S2 1·2기수 모집** (7월 중순부터)
- [ ] 첫 외부 모임 시드 ≥ 25건 누적 (어그리게이션 대시보드 자산)
- [ ] 사업자 등록 / 통신판매업 신고 절차 시작 (Phase 2 PG 도입 사전 준비)

**예상 매출**: ₩600,000~₩900,000 (졸업 후·신규 모집 사이의 dip)

**Q1 합계 매출**: ₩2,100,000~₩2,900,000 / 순수익 ~₩1,500,000~₩2,200,000

---

### Q2 (4~6개월차) — 패키지 안정화 + 멤버십 베타

**목표**: 6주 패키지를 분기당 2기수 안정 운영. 멤버십 베타 출시.

#### 4개월차 (8월) — S2 1·2기수 운영
- [ ] S2 1기수 (토요일) + 2기수 (일요일) 동시 운영
- [ ] 단발 1~2회 (S3 모집 페르소나 학습용)
- [ ] **PG 도입** (사업자 등록 완료 전제) — 토스페이먼츠 또는 포트원
- [ ] 카카오 알림톡 리마인더 도입 (24h / 2h)

**예상 매출**: ₩2,400,000~₩2,700,000

#### 5개월차 (9월) — S2 졸업 + 멤버십 베타 출시
- [ ] S2 졸업 / 데모데이 통합
- [ ] **멤버십 베타 출시** — 졸업생 한정 ₩19,000/월 (정상가 ₩29,000) 6개월 락인
- [ ] 인스타 광고 첫 시도 (소액 ₩100,000으로 4~6주 패키지 모집 광고)

**예상 매출**: ₩1,200,000~₩1,800,000 + 멤버십 ₩200,000~₩400,000

#### 6개월차 (10월) — S3 운영 + 멤버십 정상가 전환
- [ ] S3 1·2기수 운영 (10월~11월)
- [ ] 멤버십 정상가 ₩29,000/월 전환, 비졸업생도 가입 가능
- [ ] 영상강의 첫 콘텐츠 녹화 시작 (1~2주차 분량)

**예상 매출**: ₩2,400,000~₩2,800,000 + 멤버십 ₩500,000~₩900,000

**Q2 합계 매출**: ₩6,000,000~₩8,200,000 / 순수익 ~₩4,000,000~₩5,500,000

---

### Q3 (7~9개월차) — 비강사 매출 + B2B 첫 출강

**목표**: 영상강의 / 전자책 출시. B2B 첫 1건 수주.

#### 7개월차 (11월) — S3 졸업 + S4 모집 + 영상강의 1차 출시
- [ ] S3 졸업
- [ ] S4 1·2기수 모집 (12월~1월)
- [ ] **영상강의 출시 — 클래스101 또는 인프런** (6주 패키지의 1~3주차 분량 압축)
- [ ] 영상강의 가격 ₩99,000 (정상가 ₩149,000, 출시 할인)

**예상 매출**: 강의 ₩900,000 + 멤버십 ₩900,000 + 영상강의 ₩500,000~₩1,500,000

#### 8개월차 (12월) — S4 운영 + B2B 첫 영업
- [ ] S4 1·2기수 운영
- [ ] **B2B 영업 시작**: 본업 회사 + 본인 네트워크 회사 5곳에 "사내 AI 코딩 2시간 워크숍" 제안서
- [ ] 영상강의 후기 ≥ 30건 확보 후 인프런 광고 슬롯 입찰

**예상 매출**: 강의 ₩2,400,000 + 멤버십 ₩900,000 + 영상강의 ₩400,000

#### 9개월차 (1월) — B2B 첫 출강
- [ ] **B2B 1건 출강 (단가 ₩100~₩150만)**
- [ ] S4 졸업 / 데모데이 (시즌제 안정화)
- [ ] 1:1 코칭 베타 출시 (멤버십 회원 한정 월 5명)

**예상 매출**: 강의 ₩600,000 + 멤버십 ₩900,000 + 영상강의 ₩400,000 + B2B ₩1,200,000 + 1:1 ₩300,000

**Q3 합계 매출**: ₩9,000,000~₩11,000,000 / 순수익 ~₩6,000,000~₩7,500,000

---

### Q4 (10~12개월차) — 강사 1명 추가 + 커뮤니티 자율 + 탈주 결정

**목표**: 운영자 1인 천장을 깨는 첫 시도. 강사 1명 추가 또는 매니저 1명 채용. 12개월차에 탈주 여부 결정.

#### 10개월차 (2월) — 강사 후보 영입
- [ ] **6주 패키지 졸업생 중 강사 후보 1명 선정** (운영자 보조 → 토요/일요 분담)
- [ ] 강사 보수: 1기수당 ₩400,000 (운영자 ₩800,000 / 강사 ₩400,000 분배 = 정원 6명 × ₩180,000 = ₩1,080,000 매출 - 강의실 ₩30,000 × 6 = ₩180,000 - 강사 ₩400,000 = ₩500,000 운영자 순수)
- [ ] 강사 트레이닝 2주 (관찰·1회 보조 강사 → 단독 강의)

**예상 매출**: 강의 ₩2,000,000 + 멤버십 ₩900,000 + 영상강의 ₩400,000 + B2B ₩1,200,000

#### 11개월차 (3월) — 듀얼 트랙 운영
- [ ] 운영자 토요반 + 강사 일요반 (S5 1·2기수)
- [ ] 멤버십 독립 콘텐츠 ("멤버 전용 라이브 Q&A" 월 1회) 시작
- [ ] B2B 2건 수주 목표

**예상 매출**: 강의 ₩2,400,000 + 멤버십 ₩1,200,000 + 영상강의 ₩400,000 + B2B ₩2,000,000 + 1:1 ₩400,000

#### 12개월차 (4월) — 회사 탈주 의사결정
- [ ] **3개월 평균 순수익 ≥ ₩4,000,000 + 멤버십 회원 ≥ 50명 + B2B 누적 ≥ 5건 → 탈주 가능 신호**
- [ ] 탈주 결정 시: 회사에 통보, 6주~3개월 후 퇴사
- [ ] 탈주 결정 X 시: 강사 1명 채용으로 부담 분산하며 1년 더 운영

**Q4 합계 매출**: ₩9,000,000~₩12,000,000 / 순수익 ~₩6,000,000~₩8,000,000

## 4. 수원-서울 거점의 공간 전략

### 4-1. Phase별 공간
| 페이즈 | 공간 | 비용 (월) |
| --- | --- | --- |
| Phase 0~1 | 카페 / 시간제 스터디룸 (수원역·강남역 인근) | ₩30,000 × 회당 |
| Phase 2~3 | **고정 스터디룸 회원권** (수원·강남 각 1곳) | ₩200,000~₩400,000 |
| Phase 4 | **자체 작업실 / 1인 사무실** (수원 광교 또는 강남 인근 코워킹) | ₩400,000~₩800,000 |

### 4-2. "거점 2개" 의 의미
- 수원 (광교/수원역) — 본인 거주지 인근, 공간비 낮음, 동네 모객 강함
- 서울 (강남역/사당) — 직장인 모객 강함, 광역 트래픽
- **둘 중 하나만** 으로는 회당 6명 충원이 빠듯할 수 있음. 듀얼 거점이 정원 충족률을 30~50% 올림.

### 4-3. 자체 공간 도입 트리거
- 월 평균 강의 횟수 ≥ 8회 = 시간제 ₩30,000 × 8 = ₩240,000 → 고정 회원권 비용과 같아짐
- 월 평균 ≥ 12회면 자체 공간이 ROI 양수

### 5-2. 시기별 권장 분배

- **Q1**: 인스타 릴스 + 소모임/당근 + 추천 코드만. 블로그·유튜브는 후순위.
- **Q2**: + 블로그 SEO 본격 시작 (월 4편). 광고 ₩100,000/월 시범.
- **Q3**: + 유튜브 숏츠 시작 (영상강의 출시와 함께).
- **Q4**: 모든 채널 활성. 광고 ₩300,000~₩500,000/월.

### 5-3. 콘텐츠 마케팅의 디시플린
- **회차당 인스타 릴스 1편 의무** — 결과물 공개 + 후기 1줄
- **블로그 SEO 글 = 학습 라이브러리 글 = 멤버십 콘텐츠** 의 3-1 자산화 (1번 만들고 3번 활용)
- **인플루언서 협업 X** — 1년차에는 단가 안 맞고, 본인 진정성이 더 강함

## 6. 커뮤니티의 역할 — 매출이 아니라 retention의 자산

> 커뮤니티는 직접 매출 채널이 아님. **retention의 자산** 으로 보고 운영.

### 6-1. 단계별 커뮤니티 형태

| 페이즈 | 형태 | 운영자 부담 |
| --- | --- | --- |
| Phase 0~1 | 카카오 오픈채팅방 1개 | 매일 30분 |
| Phase 2 | + 강의별 단톡방 자동 생성 + 멤버십 단톡방 | 매일 30분 |
| Phase 3 | + 사이트 내 게시판/Q&A 활성화 (졸업생 디렉토리 공개) | 매일 60분 |
| Phase 4 | + 자체 데모데이 정기화 + 멘토 등급제 (졸업생 → 멘토 자율 활동) | 매일 30분 (자율 운영으로 분담) |

### 6-2. retention 핵심 메커니즘
1. **졸업 → 동기 그룹 카톡방** = 6주를 같이 한 사람들끼리 평생 친구. 친구가 있으면 다음 시즌 모집 시 자연 신청.
2. **데모데이 발표 → 인스타 자랑** = 결과물이 외부에 노출되는 순간 자존감 ↑ → 충성도 ↑.
3. **멘토 등급** = 졸업 후 신규 회원에게 답글 달면 멘토 등급 자동 승격 → 무료 1:1 코칭 1회 / 멤버십 1개월 무료 등 보상.

### 6-3. 커뮤니티가 매출로 전환되는 지점 (Q3+)
- 졸업생이 "자기 회사에 출강 의뢰" 매개 → B2B 채널의 50% 이상이 졸업생 추천에서 옴
- 멤버십 회원의 친구 추천 → 신규 가입의 30%
- 모각작 / 사이드프로젝트 매칭 → 멤버십 구독 유지의 핵심 동기

## 7. 제품 — 사이트 자체의 진화

> 1차 데모 → 백엔드 → 멤버십 시스템 → 강사 다중 시스템의 4단계.

### 7-1. 사이트 진화 단계

| 단계 | 사이트 형태 | 시점 |
| --- | --- | --- |
| v0 | 프론트만 + 더미 JSON (현재) | 5월 |
| v1 | + 백엔드 + PostgreSQL + 회원 가입 + 결제 | 7~8월 |
| v2 | + 멤버십 자동 갱신 + 라이브러리 권한 분기 + 알림톡 | 9~10월 |
| v3 | + 강사 다중 운영 (운영자 + 보조강사) + 강사별 매출 통계 | 2~3월 |
| v4 | + B2B 의뢰 폼 + 자체 데모데이 라이브 송출 | 4~5월 (탈주 후) |

### 7-2. 자체 사이트의 매출 연결성
- **"사이트 = 자동 모객 + 자동 결제 + 자동 후기 수집"** = 운영자 시간 절감의 핵심.
- 사이트 없이 12개월 매출 ₩2,800만 한계, 사이트 있으면 ₩4,000~₩5,000만까지 가능.
- ROI 명확: v1 사이트 만드는 시간 80~100시간 = 시급 ₩30,000 = ₩300만 비용 → 1년 매출 증대 ₩1,000~₩1,500만 = ROI 3~5배.

## 8. 위험 시나리오와 비상 계획

### 8-1. "모객이 안 됨" 시나리오
- 첫 단발 강의 정원 6명 중 2명 → 가격을 ₩25,000으로 낮추거나 "친구 데려오면 둘 다 무료" 추천 코드.
- 4주 연속 정원 미달 → 페르소나·시간대·장소 재검토. 매출 0인 달도 견딜 수 있어야.
- **비상 자금 = 회사 본업 월급의 6개월치** 를 사이드 프로젝트 별도 통장에 둘 것 (탈주 전 단계에서).

### 8-2. "본업 회사가 사이드 프로젝트 알게 됨" 시나리오
- 사업자 등록은 본인 명의로 가능 (회사 취업 규칙에 따라). **사전에 회사 취업규칙 확인 필수.**
- 사업자 등록 = 회사가 알 수 있는 신호. 4대보험·세무 신고 시점에 IT 자동 통보 가능.
- 회사 권리: 보통 "본업 시간 외 활동" 은 OK이나 "경업 금지" / "겸직 신고 의무" 조항 있음. 본인 회사 약관 사전 확인.
- 권고: **9~10개월차 시점**에 회사에 자진 신고 (탈주 전 1~3개월). 회사가 정중히 거절하면 그 자체가 탈주 결정의 trigger.

### 8-3. "운영자 본인 번아웃" 시나리오
- 강사 1명 추가가 가장 효과적 비상 카드. Q4의 강사 영입은 "비상 카드 미리 준비" 의 의미도 있음.
- 시즌 사이 휴식 2~3주를 모집 기간으로 두면 자연 휴식 가능.
- 비상시 6주 패키지 1기수만 운영 (월 1회), 나머지는 멤버십·영상강의 매출로 버팀.
