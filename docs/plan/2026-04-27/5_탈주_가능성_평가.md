# 리포트 5 — P6ix 탈주 가능성 평가

> **이 리포트의 질문**: "지금 시작해서 회사를 떠나도 될 만한 각이 나오나? 언제, 어떤 조건에서?"

작성일: 2026-04-27
선행 문서: 리포트 1 (진단), 리포트 2 (시장), 리포트 3 (커리큘럼), 리포트 4 (로드맵)

---

## 0. 한 줄 결론

> **지금 당장 탈주는 NO. 그러나 12개월 후 탈주는 YES (조건 충족 시).**
>
> 6~12개월간 회사 다니면서 야간/주말로 매출 트랙 레코드를 만들고, 다음 트리거가 모두 충족된 시점에 탈주가 정답:
> 1. 3개월 평균 월 순수익 ≥ ₩400만
> 2. 멤버십 회원 ≥ 50명 (월 정기 매출 ≥ ₩100만)
> 3. B2B 출강 누적 ≥ 5건 (반복 의뢰처 ≥ 1)
> 4. 비상 자금 ≥ 회사 월급의 6개월치 (탈주 후 6개월 무수입 버틸 수 있음)
> 5. 본인 정신·체력 상태 OK (번아웃 없음)
>
> 5개 중 4개+ 충족 = 탈주 GO. 3개 이하 = 6개월 더 버티기.

## 1. 왜 "지금 당장" 은 NO인가

### 1-1. 객관적 이유

1. **매출 트랙 레코드가 0**
   - 첫 강의를 아직 한 번도 안 굴렸음 (소모임 앱 캡쳐만 보고 있음).
   - 시장이 있다는 가설은 강하지만, **본인 페르소나·가격·운영 능력이 그 시장을 잡는지** 는 검증 0.
   - 사업이라 부르려면 최소 3~6개월 운영 데이터가 필요.

2. **매출 천장 미확인**
   - 1인 강사로 월 ₩300~₩500만 가능한 구조는 시장 분석에서 가능성 높음. 그러나 **본인이 그 천장에 도달할 수 있는 강사인지** 는 모름.
   - 첫 1~3회 강의에서 정원 충족률·후기·재신청 의향 데이터를 보고 천장 추정 가능.

3. **네트워크 자산 0**
   - B2B 출강은 1년차 매출의 30~40%를 차지하는 핵심 채널 (리포트 4). 이건 강의 받은 사람이 → 자기 회사에 추천하는 구조로 형성됨.
   - 졸업생 100명 누적 시점이 B2B 매출이 본격 오는 시점.
   - 0명인 지금 탈주하면 B2B 채널이 6~12개월 동안 0.

4. **운영 디시플린 검증 X**
   - 콘텐츠 만들기·강의·마케팅·운영(환불·CS)·세무·가족과의 시간배분 — 1인 운영자는 이 모든 걸 조절해야 함.
   - 회사 다니면서 굴려보면 어느 부분에서 무너지는지 미리 알 수 있음. **회사 안정망 없이 무너지면 회복이 어려움.**

### 1-2. 심리적 이유

1. **"회사 그만두면 잘할 것 같다" 의 환상**
   - 본업 시간 8시간이 빠지면 강의에 그 시간을 다 쓸 것 같지만 실제론 30~50%만 강의/콘텐츠로 가고 나머지는 다른 곳으로 빠짐 (업계 통설).
   - 회사 그만두면 자연스럽게 매출 2배 되지 않는다. 마케팅 시간을 늘리면 매출이 늘긴 하지만 비례 ↑ 가 아님.

2. **고정 수입의 심리적 가치**
   - 매월 정해진 날 통장에 들어오는 월급은 "사업 매출의 변동성" 을 받쳐주는 정신 안정망.
   - 안정망이 없으면 첫 모객 부진의 1~2개월에 패닉 매수(가격 인하·무리한 광고)가 일어남.

3. **회사를 "탈주" 가 아닌 "졸업" 으로 봐야**
   - "탈주" 의 어감은 도망. 그러면 사업도 "도망친 결과" 가 됨.
   - "졸업" 의 어감은 다음 단계로의 전진. 사업이 본업의 자연 확장이 되도록.
   - 이 framing이 1년차 정신상태를 결정함.

## 2. 왜 "1년 후" 는 YES인가 (가능성)

### 2-1. 시장 타이밍
- 2026~2028년이 황금 분기 (리포트 2). 1년 후도 여전히 분기 안.
- 1년 늦으면 ① 클래스101 진입 ② AI 도구 한국 공식 교육 등 경쟁 추가 가능. 그러나 어그리게이션 대시보드와 오프라인 락인 자산이 1년 누적되어 있으면 후발주자 진입 어렵게 만듦.

### 2-2. 자산 누적
- 12개월간 졸업생 ≥ 100명 (시즌별 6명 × 2기수 × 4시즌 + 단발 12회)
- 후기 캡쳐 ≥ 50건
- 영상강의 1~2편 출시 (수동 매출 채널)
- 인스타 팔로워 ≥ 1,500
- B2B 출강 누적 5~10건
- 외부 모임 시드 ≥ 100건 (어그리게이션 자산)

이 자산들이 다 모이면 **"강사 1인이 만든 가능성" 에서 "지속 운영 가능한 사업 시스템" 으로 전환** 됨. 탈주의 객관적 근거가 됨.

### 2-3. 매출 안정성
- 12개월차에 월 매출 ₩600~₩800만 / 순수익 ₩400~₩500만이 보수~중간 시나리오.
- 본업 월급 (대기업 기준 ₩400~₩600만 추정) 과 비슷하거나 살짝 위.
- **순수익 ≥ 본업 월급 × 1.0** 이 탈주 최소 조건. **× 1.5** 이면 안정. **× 2.0** 이면 자축.

### 2-4. 운영 자동화 검증
- 12개월간 강의·결제·환불·후기·다음 회차 모집의 흐름이 사이트와 운영 룰로 자동화되어 있음.
- 하루 운영자 투입 시간이 4~6시간 정도로 안정.

## 3. 탈주 트리거 5가지 — 정량 기준

| # | 트리거 | 측정 방법 | 충족 시 의미 |
| --- | --- | --- | --- |
| 1 | **3개월 평균 월 순수익 ≥ ₩400만** | 연속 3개월 평균. 일시적 스파이크 X | 최소 생계 + 사업 재투자 가능 |
| 2 | **멤버십 회원 ≥ 50명, 월 정기 매출 ≥ ₩100만** | 멤버십 결제 시스템 통계 | 변동성 낮은 베이스 매출 확보 |
| 3 | **B2B 출강 누적 ≥ 5건, 반복 의뢰처 ≥ 1** | 청구서/세금계산서 발행 건 수 | 1인 천장 깨는 채널 검증됨 |
| 4 | **비상 자금 ≥ 회사 월급 × 6개월치** | 별도 통장 잔액 | 무수입 6개월 버팀목 |
| 5 | **번아웃 없음 + 가족·파트너 동의** | 본인 자가 진단 + 가족 컨센서스 | 정신 안정망 |

### 3-1. 트리거 충족 시점별 권고

- **5개 모두 충족** → 탈주 GO. 1~3개월 내 회사에 통보, 6주~3개월 후 퇴사.
- **4개 충족** → 누락된 1개에 따라 다름:
  - #1, #2 미충족 → 6개월 더 버티기. 매출 안정화 우선.
  - #3 미충족 → 탈주 가능. B2B는 탈주 후 시간 늘려 빠르게 채울 수 있음.
  - #4 미충족 → **탈주 NO**. 비상 자금이 없으면 첫 부진에 패닉.
  - #5 미충족 → **탈주 NO**. 정신 무너지면 사업 무너짐.
- **3개 이하 충족** → 탈주 NO. 1년 더 본업 + 사이드.

## 4. 비상 자금 — 얼마가 필요한가

### 4-1. 탈주 후 6개월 무수입 시뮬레이션

> 가정: 본업 월급 ₩500만, 사업 월 매출 ₩500만이 있으나 변동성 큼 (0~1,000만).

| 항목 | 월 비용 | 6개월 |
| --- | --- | --- |
| 주거 (월세 또는 대출이자) | ₩800,000 | ₩4,800,000 |
| 식비 | ₩600,000 | ₩3,600,000 |
| 교통/통신 | ₩200,000 | ₩1,200,000 |
| 보험 | ₩150,000 | ₩900,000 |
| 사업 운영비 (대관·광고·도구) | ₩500,000 | ₩3,000,000 |
| 세금 (사업소득) | ₩400,000 | ₩2,400,000 |
| 비상 (의료·가족) | ₩300,000 | ₩1,800,000 |
| **합계 6개월** | ₩2,950,000 | **₩17,700,000** |

> **권고**: 비상 자금 = ₩2,000만 (약 6~7개월). 본인 회사 월급 × 4~6 정도가 안전선.
> 미달이면 매월 별도 통장에 ₩100만씩 적립하는 디시플린 필요.

### 4-2. 비상 자금이 사라질 시점에 매출이 회복 안 된다면?
- 매출 회복 안 되는 데 6개월 → 사업 모델 자체에 문제. 사업 정리 + 재취업.
- 이 시나리오의 가능성 = 30% (시장 분석에서 사업 성공 가능성 70% 추정).
- 따라서 **재취업 가능성도 항상 열어둘 것**. 탈주 시점에 기존 회사와 좋은 관계로 마무리 + 동종업계 연락처 유지.

## 5. 회사 본업 vs 사이드의 균형 — 탈주 직전 1년

### 5-1. 시간 배분 (주 단위, 회사 다닐 때)

| 활동 | 본업 시간 | 사이드 시간 | 합계 |
| --- | --- | --- | --- |
| 회사 본업 | 40~45h | 0 | 40~45h |
| 강의 운영 (주 1~2회) | 0 | 4~6h | 4~6h |
| 콘텐츠 제작 (슬라이드/영상/블로그) | 0 | 6~10h | 6~10h |
| 마케팅 (인스타·블로그·DM 응대) | 0 | 4~6h | 4~6h |
| 사이트 개발/운영 | 0 | 3~5h | 3~5h |
| **사이드 합계** | | **17~27h/주** | |
| 잠/식사/이동/가족 | | | 60~70h |
| **총 합계** | 40~45 | 17~27 | **57~72h** 활동 |

> 주 60~70시간 활동 = 보통의 **풀타임 + 사이드 프로젝트** 의 한계. 6개월~1년이면 OK, 그 이상 = 번아웃.

### 5-2. "사이드 시간을 줄여야 하는 신호" 5가지

1. 강의 회차 직전에 콘텐츠를 새벽 2~3시까지 만들고 있음 (주 1회 이상)
2. 본업 회의에서 사이드 일이 머릿속에 떠남
3. 가족·파트너와 갈등 발생 빈도 ↑
4. 운동·식사 거르는 빈도 ↑
5. 잘 자고 깬 뒤에도 피로감

3개 이상 = 사이드 줄이기. 5개 모두 = 즉시 1~2주 휴식.

### 5-3. 회사에 자진 신고 시점

- **9~10개월차** (탈주 결정 1~3개월 전).
- 신고 시 톤: "사이드 프로젝트로 ◎◎ 모임을 운영 중. 본업과 충돌 없음. 만일 회사 정책상 문제 시 알려달라."
- 회사 반응:
  - "OK, 본업만 잘 하면 됨" → 좋음. 그대로 유지.
  - "겸직 금지 조항 위반. 정리하든지 퇴사하든지 결정" → 탈주 결정의 자연 trigger.
  - "사이드는 양해. 단 회사 자원 활용 X" → 룰 명확화. OK.

## 6. "회사로 안 돌아갈 자신이 없으면 탈주하지 말 것"

> 1인 사업자의 가장 큰 함정 = "정신적 단절" 의 어려움.

### 6-1. 회사 vs 사업의 정신 모드 차이

| 회사 | 1인 사업 |
| --- | --- |
| 정해진 출퇴근 | 24시간 자기 결정 |
| 동료가 있음 | 혼자 |
| 역할이 정해짐 | 모든 역할을 본인이 |
| 매출/회계는 회사 일 | 매출/회계가 자기 일 |
| 평가가 명확 | 평가는 매출 숫자 + 인스타 팔로워뿐 |
| 인풋 = 시간 | 인풋 = 시간 + 마케팅 + 인지도 |

### 6-2. 6개월 후 "회사가 그리워질" 가능성
- 1인 사업 시작 후 3~6개월차에 60~70%의 사람이 "회사가 그립다" 느낌.
- 이 시점에 매출이 약하면 패닉 결정 (가격 인하, 무리한 광고).
- **권고**: 탈주 1개월차에 "이 결정을 후회해도 1년은 버티겠다" 의 약속을 본인에게 적어두기. 노트에 손으로.

### 6-3. 사업 실패 시 회복 시나리오
- 12개월 운영해도 트리거 미충족 → 사업 종료, 재취업.
- 재취업 가능성 = 본인 본업 분야의 시장 + 12개월 사이드 운영 경험.
- "코딩 사이드 모임 12개월 운영 + 사업자 등록 + 영상강의 1~2편" 의 이력은 **재취업 시 마이너스 아니라 플러스** (개발+사업+교육 융합 인재).
- 따라서 탈주 후 사업 실패도 회복 가능. 그러나 회복까지 6~12개월 무수입 시나리오는 가족과 사전 합의 필수.

## 7. 탈주 시기 후보 — 시나리오별

### 7-1. 시나리오 A — 보수 (12개월 후, 2027년 4월)
- 6개월차 (10월)에 트리거 5개 중 2~3개 충족 시작.
- 12개월차 (4월)에 4~5개 충족. 5월부터 회사에 통보 → 7월 퇴사.
- **권장도 ★★★★★** — 가장 안전, 가장 높은 성공 확률.

### 7-2. 시나리오 B — 중간 (8~10개월 후, 2026년 12월~2027년 2월)
- 8~10개월차에 트리거 5개 중 4개 충족 (특히 #1 매출, #4 비상자금).
- 회사 보너스/연말 정산 후 떠나는 타이밍 (12월~2월).
- **권장도 ★★★** — 매출 안정성이 12개월보다 약함. 본인 자신감과 가족 컨센서스 필요.

### 7-3. 시나리오 C — 공격 (6개월 후, 2026년 10월)
- 6개월차에 매출 ₩400만 도달 + 비상자금 모두 충족.
- 본업 회사가 사이드 알아서 통제 시작 → 자연 trigger.
- **권장도 ★** — 매출 트랙 레코드 부족. 첫 모객 부진 시 회복 어려움. 비상 시나리오일 때만.

### 7-4. 시나리오 D — 지연 (18개월 후, 2027년 10월)
- 12개월차에 트리거 미충족 → 본업 유지 + 사이드 6개월 더.
- 강사 1명 추가로 운영 부담 분산.
- 18개월차에 트리거 다시 평가.
- **권장도 ★★★★** — 트리거 미충족 시 정답. "안 떠나도 사업은 굴러간다" 의 자신감이 진짜 탈주 신호.

## 8. 탈주 후 첫 30일 권고 (탈주 결정 시)

### 8-1. 첫 1주
- 출근 안 하는 첫 월요일 = 정신 혼란 강함. 의도적 휴식 1주 (강의도 X).
- 가족·파트너와 시간.
- 새벽~오전 루틴 만들기 (운동, 명상, 글쓰기).

### 8-2. 첫 2~4주
- 강의 회차 늘리기 (주 2회 → 주 3회). 모객은 인스타·블로그로.
- 영상강의 콘텐츠 본격 제작 (시간 ↑).
- B2B 영업 메일 발송 ≥ 20곳.
- 멤버십 콘텐츠 월 1회 라이브 Q&A 시작.

### 8-3. 첫 90일 KPI
- 월 매출 ≥ ₩600만 (탈주 전 평균보다 ↑, 시간이 늘었으니까)
- 멤버십 회원 ≥ 70명
- B2B 1건 추가 수주
- 번아웃 없음

3개월 후 KPI 미달 시 강사 1명 채용 또는 콘텐츠 외주 검토.

## 9. 본인과의 약속 — 탈주 결정의 자가 점검 5문항

탈주 결정 직전 본인에게 솔직하게 답할 것:

1. **회사가 갑자기 좋은 승진/연봉 인상 제시해도 떠날 거야?** (Yes 면 진짜 떠날 준비 됨)
2. **6개월 매출 0이어도 후회 안 해?** (Yes 면 사업 정신 OK)
3. **12개월 후 매출 ₩300만 (현재의 60%)이어도 1년 더 버틸 거야?** (Yes 면 끈기 OK)
4. **가족·파트너가 진심으로 동의했나, 아니면 내가 설득한 거야?** (진심 동의 = OK, 설득 = Risk)
5. **회사가 아닌 시장에서 "내 가치" 가 ₩500만/월 이상이라고 객관적으로 증명된 거야?** (증명 = 매출 트랙 레코드)

5개 모두 Yes = 탈주 GO. 1개라도 No = 6개월 더.
