## 결정 필요한 것 (시안 보고 답해줘)

1. **2주차 버튼 라벨** — "2주차 준비중 (5/8 공개 예정)" vs "2주차 미리보기 →" (지금은 disabled). 어느 쪽?
2. **신청 CTA 클릭 동작** — `#apply` 앵커로 페이지 내 신청 폼? 아니면 카톡 오픈채팅 외부 링크? 아직 결제·신청 폼 미구현이면 카톡 외부 링크로 시작이 안전.
3. **Footer 라이선스 문구** — "CC BY 4.0" 표기 OK? 아니면 "출처 표기 시 자유 사용" 자연어만 쓸지?
4. **운영자 SNS** — 인스타·카톡·이메일 핸들 알려주면 시안에 채워서 그대로 적용. 아니면 placeholder로 두고 본인이 나중에 직접 채울지?

이 4개만 답해주면 시안대로 바로 코드 들어갈게.
