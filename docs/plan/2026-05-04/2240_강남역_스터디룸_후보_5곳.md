## 환불·취소 정책 — 중요

### 스페이스클라우드 공통 정책
- **결제 후 2시간 이내**: 100% 환불
- **그 외**: 호스트가 자체 설정한 환불률 적용 → 룸별로 다름

### 운영자가 직접 확인할 것
1. 위 5곳 중 한 곳 클릭 → 상세 페이지 하단 **"이용 안내 / 환불 정책"** 박스
2. 호스트가 박아둔 환불률 표 확인 (예: "이용 7일 전 100% / 3일 전 70% / 1일 전 30%")
3. **1일 전 50% 이상 환불**되는 곳만 후보로 남김 — 노쇼 시 본인 부담 한도

## 추천 전략 2가지

### A안 — 스페이스클라우드 "공간이즈" 또는 "모임플러스"
- 강남역 5번 출구 1분 → 처음 오는 사람도 못 헷갈림
- 일요일 오후(14~17시)는 인기 슬롯 → **지금 잡는 게 유리**
- 결제 전 환불정책 표 스크린샷 보관

### B안 — 노블스터디룸 (24시간 운영, 백업)
- 강남역 11번 출구, 일요일 시간 제약 0
- 인당 가격제(1,700원/시간/인) → **6명 × 3시간 ≈ 30,600원**
- 단점: 인당 가격제 → 사전 인원수 변경 가능 여부 호스트에게 확인 필요
