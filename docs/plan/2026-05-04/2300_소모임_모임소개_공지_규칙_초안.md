### 작성 원칙 (이거 지키면 더 안 부풀음)

1. **같은 말 두 번 하지 않기** — "두 시간 원데이"가 위·아래 두 번 나오면 한 번 지움
2. **이력·운영자 소개 X** — 그건 사이트(`/about`)에 보내고 모임 소개에서는 빼기
3. **이모지는 섹션 헤더만** — 본문 줄마다 박으면 글자 수 폭증
4. **링크는 풀 URL 한 번만** — `vibe.me.kr` 한 곳에만 박고 나머지는 슬러그(`/guide`)
5. **"이런 분께 잘 맞아요" 항목 3개 → 1줄로 압축** — `🙋 코딩 처음이신 분도 환영해요.`
6. **인사말 / 마무리 인사 X** — 소모임 앱은 첫 인상이라 정보 밀도 우선

   - 6개월 이상 미접속 시 자동 정리될 수 있어요

3. 호칭
   - 닉네임 + "님" 통일 (모임장도 똑같이 "명관님")

   - 입금 마감: 모임 시작 72시간 전까지
   - 환불 가능: 모임 시작 48시간 전까지 100% 환불
   - 그 이후 (48시간 이내 ~ 시작 후 / 노쇼): 환불 불가

규칙은 함께 다듬어 갑니다. 의견은 자유게시판에 남겨 주세요.

> 원본의 "3달에 한 번 이상 모임 참가" → "6개월 미접속 시 자동 정리" 로 부담 낮춤.
> 환불 규정은 운영자 정책으로 확정 (입금 72h 전 / 환불 48h 전).

## 7. 주의사항

- **원본 모임장(강준현) 모임명 그대로 쓰지 말 것** — 분쟁 소지
- 이모지는 소모임 앱에서는 OK (사이트 본문은 글로벌 룰상 금지지만, 외부 플랫폼 글은 다른 톤)
- 사이트 URL은 항상 `https://www.vibe.me.kr` 풀 도메인으로 표기 (소모임 앱 내 링크 클릭 가능)
- 첫 1~2회는 **무료 또는 1만원 이하**로 진입장벽 낮춰 멤버 끌어모은 뒤 정상가 전환

### 커리큘럼 공개 정책 운영 메모

- **톤 원칙**: "우리는 다르다" 식 비교/공격 표현은 모든 글에서 금지. 그냥 "있어요 / 보세요 / 같이 해봐요" 권유 톤만.
- **"커리큘럼만 보고 안 오는 사람" 은 손해 아님** — 그 사람은 어차피 안 올 사람. 대신 "이 정도 정리해 두는 사람이면 믿을 만하다" 신뢰를 다른 사람에게 쌓아 줌.
- 사이트 가이드 페이지(`/guide`)는 **항상 최신 회차 기준으로 유지**. 가이드와 실제 수업 내용이 달라지면 신뢰 깨짐.
- 자료 도용 방지를 위해 워터마크 또는 운영자 시그니처 박는 거 검토 (`agents/ui/operator-profile-module.md` 참고).
- 카톡방 운영: "가이드 보다 막힌 분" 의 질문이 올라오면 **그 자리에서 한 줄 답 → 다음 원데이 클래스 자연스럽게 안내** 의 흐름.
- 단어 선택: 외부 노출 글에서는 "교안" 보다 **"커리큘럼" 또는 "가이드"** 우선 (학원 느낌 약하게).
