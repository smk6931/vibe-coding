# 프론트엔드 기본 용어

## Hero

Hero는 페이지에 들어왔을 때 사용자가 가장 먼저 보는 대표 소개 영역이다.

보통 포함되는 것:
- 페이지가 무엇인지 알려주는 큰 제목
- 짧은 설명 문장
- 주요 CTA 버튼
- 경우에 따라 대표 이미지나 검색/필터 영역

이 프로젝트에서 `/`의 hero는 큰 배너 이미지가 아니라 다음 영역을 뜻한다.

```text
지금 이 동네에서, 뭐 하지
수원·서울 라이트 클래스 + 수도권 외부 모임을 지도·카드로 한눈에.
```

그리고 문서에서 `[히어로 + 필터]`라고 쓰면, 이 대표 문구 영역과 날짜/지역/가격/자체·외부 필터를 묶어서 말한다.

중요한 이유:
- 첫 화면에서 사이트의 목적을 즉시 전달한다.
- 모임 탐색 페이지에서는 사용자가 바로 필터링하거나 카드 목록으로 내려갈 수 있게 만든다.
- 랜딩페이지처럼 과하게 꾸미기보다, 실제 모임을 찾는 흐름을 방해하지 않는 것이 중요하다.

## SEO Meta

SEO meta는 검색엔진과 브라우저가 페이지를 이해하도록 HTML `<head>`에 넣는 정보다.

대표 항목:
- 페이지 제목: `title`
- 페이지 설명: `description`
- 검색 노출용 키워드나 canonical URL

예시:

```html
<title>바이브 세션 - 바이브코딩 원데이 클래스</title>
<meta name="description" content="수원·서울에서 2시간으로 시작하는 바이브코딩 입문 클래스" />
```

이 프로젝트에서 중요한 이유:
- `vibe.me.kr` 도메인을 검색하거나 공유했을 때 신뢰감 있는 제목과 설명이 필요하다.
- 소모임, 인스타, 카카오 오픈채팅 등 외부 유입에서 사이트 정체성이 명확해야 한다.

## OG Tag

OG는 Open Graph의 줄임말이다. 카카오톡, 페이스북, 디스코드, 슬랙 같은 곳에 링크를 공유했을 때 뜨는 미리보기 카드를 제어한다.

대표 항목:

```html
<meta property="og:title" content="바이브 세션" />
<meta property="og:description" content="수원·서울, 2시간으로 끝내는 바이브코딩 라이트 클래스" />
<meta property="og:image" content="https://vibe.me.kr/images/og.png" />
<meta property="og:url" content="https://vibe.me.kr" />
```

사용자가 `https://vibe.me.kr`을 카톡에 붙여넣으면 대략 이런 카드가 뜨게 만드는 역할이다.

```text
[대표 썸네일]
바이브 세션
수원·서울, 2시간으로 끝내는 바이브코딩 라이트 클래스
https://vibe.me.kr
```

이 프로젝트에서 중요한 이유:
- 카카오 오픈채팅, 소모임 앱, 인스타 프로필에서 링크를 공유할 때 첫인상을 만든다.
- 원데이 클래스 홍보 링크가 개인 링크처럼 보이지 않고 사이트처럼 보이게 한다.
- `og:image`는 썸네일 역할을 하므로 강의 분위기와 신뢰감을 동시에 줘야 한다.

## OG Image

OG image는 링크 미리보기 카드에 뜨는 대표 이미지다.

권장:
- `public/images/og.png` 같은 정적 경로에 둔다.
- 절대 URL로 지정한다: `https://vibe.me.kr/images/og.png`
- 작은 카드에서도 알아볼 수 있게 큰 글자, 단순한 구도, 높은 대비를 쓴다.

주의:
- 카톡은 OG 캐시가 남아서 이미지를 바꿔도 바로 반영되지 않을 수 있다.
- 너무 많은 텍스트를 이미지 안에 넣으면 모바일 공유 카드에서 읽히지 않는다.
