# 개발 지식 노트

바이브코딩 수업 준비 중 나온 프론트엔드, 백엔드, 배포, 운영 용어를 짧게 정리하는 폴더.

목적:
- 운영자가 수시로 다시 볼 수 있게 한다.
- 추후 강의 자료로 재사용할 수 있게 한다.
- AI 에이전트가 용어 설명을 했을 때 같은 맥락의 지식을 이 폴더에 남긴다.

작성 규칙:
- 파일명은 주제별 kebab-case 영어 또는 짧은 한글 사용.
- 한 문서에는 비슷한 범주의 용어만 모은다.
- 설명은 “무엇인가 / 왜 중요한가 / 이 프로젝트에서는 어디에 쓰나” 순서로 적는다.
- 실제 파일이나 URL이 있으면 예시를 함께 적는다.
