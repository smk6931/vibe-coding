# 배드민턴 대시보드 구축 과정 기록
> 비전공자 가이드용 예행연습 — 실제 진행 순서대로 기록

---

## 완성 화면 구성

| 페이지 | 경로 | 내용 |
|---|---|---|
| 메인 대시보드 | `/` | 팀 성적, 선수 컨디션 신호등, 다음 경기 일정 |
| 선수 상세 | `/player/:id` | 컨디션 추이 그래프, 능력치 레이더 차트, 최근 경기 |
| 경기 분석 | `/match` | 경기별 통계, 타임스탬프 메모 추가 기능 |

---

## STEP 1 — 프로젝트 생성

### 터미널에 순서대로 입력

**1. React + Vite 프로젝트 생성**
```bash
npm create vite@latest badminton-dashboard -- --template react
```

> `badminton-dashboard` 라는 폴더가 생성됩니다.

**2. 프로젝트 폴더로 이동**
```bash
cd badminton-dashboard
```

**3. 기본 패키지 설치**
```bash
npm install
```

**4. 추가 라이브러리 설치** (차트, 스타일, 페이지 이동)
```bash
npm install tailwindcss @tailwindcss/vite recharts react-router-dom
```

| 라이브러리 | 역할 |
|---|---|
| tailwindcss | 디자인 (다크모드, 색상, 간격) |
| @tailwindcss/vite | Tailwind를 Vite와 연결 |
| recharts | 라인 그래프, 레이더 차트 |
| react-router-dom | 페이지 이동 (대시보드 ↔ 선수 상세) |

---

## STEP 2 — 설정 파일 수정

### `vite.config.js` 수정
Tailwind를 Vite 플러그인으로 연결합니다.

```js
import { defineConfig } from 'vite'
import react from '@vitejs/plugin-react'
import tailwindcss from '@tailwindcss/vite'

export default defineConfig({
  plugins: [react(), tailwindcss()],
})
```

### `src/index.css` 전체 교체
기본 CSS를 지우고 Tailwind 한 줄로 교체합니다.

```css
@import "tailwindcss";
```

## STEP 4 — 데이터 구조 (`src/data/players.js`)

백엔드 없이 JavaScript 파일에 데이터를 직접 넣습니다.

```js
export const players = [
  {
    id: 1,
    name: "김민준",
    position: "단식",
    age: 24,
    status: "good",       // good | caution | danger
    condition: 88,        // 0~100 컨디션 점수
    stats: {
      smashRate: 82,      // 스매시 성공률
      serveAccuracy: 75,  // 서브 정확도
      netPlay: 70,        // 네트 플레이
      defense: 78,        // 수비
      stamina: 85,        // 체력
      mental: 80,         // 멘탈
    },
    conditionHistory: [   // 주간 컨디션 추이 (그래프용)
      { week: "4/14", score: 88 },
      ...
    ],
    recentMatches: [      // 최근 경기 결과
      { date: "4/14", opponent: "한국체대", result: "승", score: "21-15, 21-18" },
      ...
    ],
  },
]
```

> **핵심:** 이 파일의 데이터만 바꾸면 화면에 바로 반영됩니다.  
> 나중에 백엔드를 연결할 때도 이 구조를 그대로 씁니다.

## STEP 5 — 라우터 연결 (`main.jsx`, `App.jsx`)

### `main.jsx` — BrowserRouter 추가
```jsx
import { BrowserRouter } from 'react-router-dom'

createRoot(document.getElementById('root')).render(
  <StrictMode>
    <BrowserRouter>
      <App />
    </BrowserRouter>
  </StrictMode>,
)
```

### `App.jsx` — 페이지 경로 연결
```jsx
<Routes>
  <Route path="/"          element={<Dashboard />} />
  <Route path="/player/:id" element={<PlayerDetail />} />
  <Route path="/match"     element={<MatchAnalysis />} />
</Routes>
```

## STEP 6 — 개발 서버 실행

```bash
npm run dev
```

터미널에 아래처럼 나오면 성공:
```
VITE ready in xxx ms
➜  Local: http://localhost:5173/
```

브라우저에서 `http://localhost:5173` 열면 화면 확인 가능합니다.

> **맥북 단축키:** 터미널에서 `Ctrl+C` 누르면 서버 종료

## 자주 발생하는 오류

| 오류 | 원인 | 해결 |
|---|---|---|
| `Cannot find module 'recharts'` | 패키지 설치 안 됨 | `npm install recharts` |
| 화면이 하얗게만 나옴 | 라우터 빠짐 | `main.jsx`에 BrowserRouter 확인 |
| Tailwind 색상이 적용 안 됨 | vite.config.js 설정 누락 | tailwindcss 플러그인 추가 확인 |
| 포트 5173이 이미 사용 중 | 다른 서버 실행 중 | `npm run dev -- --port 5174` |
